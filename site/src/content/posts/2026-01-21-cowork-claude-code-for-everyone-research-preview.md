🚀 **Cowork: Claude Code for Everyone (Research Preview)**

⚠️ **macOS only** | **Team plan access: Friday, Jan 23rd**

Anthropic launched **Cowork** — a new way to work with Claude that brings the Claude Code experience to **non-developers.**
**Key highlights:**
• **Folder access**: Give Claude access to a folder on your Mac, and it can read, edit, or create files autonomously
• **Example use cases**: Reorganize downloads, create spreadsheets from screenshot receipts, draft reports from scattered notes
• **Parallel work**: Queue up multiple tasks and Claude works through them — feels like leaving messages for a coworker
• **Power-ups**: Works with Connectors, Skills, and Claude in Chrome for browser-based tasks
• **You stay in control**: Claude only accesses folders you permit and asks before significant actions
