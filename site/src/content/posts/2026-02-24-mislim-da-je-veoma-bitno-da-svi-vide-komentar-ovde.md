Mislim da je veoma bitno da svi vide @Nebojsa Lazic komentar ovde:

Svaka mu je na mestu.
A True Software Engineer:
• does not write code
• does not review the code (there are other ways for verifying agent output that manual code review)
• makes no difference between frontend, backend, mobile, web, desktop...
• does not care about the programming language - they know which language is good for a given task, and stop there: they do not have to be masters in that langauge. They can work with agents in any programming language, even the ones they have never worked before
• is aware that, in certain domains, agent is simply better/faster/smarter.
