# Commit Helper

Git workflow commands for creating conventional commits and generating PR descriptions.

## Commands

### `/commit-helper:commit`

Analyzes your staged (or unstaged) changes and creates a well-structured commit following [Conventional Commits](https://www.conventionalcommits.org/):

```
/commit-helper:commit
```

### `/commit-helper:pr-describe`

Generates a pull request description from the current branch's commits:

```
/commit-helper:pr-describe
```

## Install

```bash
/plugin install commit-helper@vicert-marketplace
```
