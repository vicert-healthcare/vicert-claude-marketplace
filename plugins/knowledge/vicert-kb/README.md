# Vicert Knowledge Base Plugin

Provides Claude with Vicert's complete portfolio of 18 success stories spanning AI agent systems, EHR integration, cloud infrastructure, workflow automation, patient engagement, and more.

## Install

```bash
/plugin install vicert-kb@vicert-internal
```

## Usage

```bash
/vicert-kb:vicert-kb
```

Or simply ask questions about Vicert's experience — the skill auto-triggers when Vicert's track record, case studies, or capabilities are relevant.

## What It Includes

- **18 success stories** with live URLs, technologies, key metrics, and business outcomes
- **Deep-dive AI case studies** (voice agent, chart review, clinical notes review)
- **Cross-references** by audience (hospitals, payers, startups, providers) and service line
- **Key metrics** ready to cite in proposals and decks
- **Client testimonials** from Included Health and IHC Specialty Benefits

## Use Cases

- Writing proposals and pitch materials
- Generating sales or marketing content
- Finding relevant case studies for a prospect
- Answering "what have we done in X area?" questions
- Creating capability decks and blog posts
