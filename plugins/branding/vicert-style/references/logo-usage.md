# Vicert Logo Usage Guidelines

## Logo File

The official Vicert logo is at:
`${CLAUDE_PLUGIN_ROOT}/references/vicert-logo.png`

PNG format, 630x172 pixels (~3.7:1 aspect ratio). Extracted from the official Vicert_Template.dotx.

## Logo Structure

The logo consists of two elements:
1. **Icon** — A circular emblem with concentric organic curves in Vicert cyan (#0099FF)
2. **Wordmark** — "VICERT" text in near-black

Both elements are part of a single image rendered as a horizontal lockup (icon left, wordmark right).

## Placement Rules

### Word / DOCX Documents
- Place the logo in the **document header**, anchored **top-right** of the column (as defined in the template)
- Template default width: **~1.3 inches** (1633538 EMUs / cx) in the header
- For title pages: **2.5–3.5 inches** centered or top-right
- Maintain aspect ratio — do not stretch or compress
- Minimum clear space: 0.25 inches on all sides
- The logo should appear on every page via the header

### Presentations
- **Title slide:** Logo top-right or centered, width ~30–40% of slide width
- **Content slides:** Logo in header or footer, width ~15–20% of slide width
- Use on white or light backgrounds for best contrast

### PDF (from Word)
- Follow the Word placement rules — the logo carries through from the header when exporting to PDF

## Color Contexts

| Background | Result | Notes |
|---|---|---|
| White / Light | Use as-is | Cyan icon + dark wordmark — ideal |
| Light gray (#EEECE1) | Use as-is | Good contrast |
| Dark backgrounds | Avoid | Wordmark becomes invisible |
| Blue / Cyan backgrounds | Avoid | Icon blends with background |

## Do Not

- Rotate or skew the logo
- Change the icon color
- Place the logo on a busy, dark, or low-contrast background
- Scale below 0.8 inches width (legibility threshold)
- Add effects (drop shadows, glows, outlines) to the logo
- Separate the icon from the wordmark
- Replace the template PNG with a different version without updating the template
